### Prerequisites

### Setup

1.  Clone the Repo with:

```
git clone https://github.com/maths29/Capstone-project-1.git
```

2. switch to the cloned directory

```
cd capstone project-1
```

### Installation

1. Install the project with it's depenedcies

